 User Authentication Module

A beginner-friendly **User Authentication Module** built using Python Flask as part of an internship task.

  Features
- User Registration
- User Login
- Change Password
- Edit & Update Profile
- Secure Password Hashing

 Technologies Used
- Language: Python
- Framework: Flask
- Database: SQLite
- ORM: Flask-SQLAlchemy

 How to Run
1. Activate virtual environment  
   `venv\Scripts\activate`
2. Install packages  
   `pip install -r requirements.txt`
3. Run app  
   `python app.py`
4. Open browser  
   `http://127.0.0.1:5000`
